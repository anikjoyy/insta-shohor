How does Javascript Work?

Javascript is a single-threaded programming language. It can do one thing at a time and it execute the code from the top to bottom. Javascript has a single call stack. The call stack is a data structure that records wherein the program are. In Javascript we push code or function on the top of the stack. So, Javascript works by executing one thing at a time.
